Unsere Module

Modul 1: Erste Schritte mit Python

Lerne die Grundlagen der Programmierung mit Python. Wir behandeln Variablen, Datentypen, Operatoren und einfache Kontrollstrukturen.

  • Installation von Python und Entwicklungsumgebung
  • Syntax und grundlegende Konzepte
  • Einfache Programme schreiben

Modul 2: HTML & CSS für Webseiten

Erstelle deine erste Webseite! Verstehe, wie HTML die Struktur definiert und CSS das Aussehen gestaltet.

  • HTML-Struktur und Tags
  • CSS-Selektoren und Styling
  • Responsive Design mit Flexbox und Grid

Modul 3: JavaScript Interaktivität

Bringe deine Webseiten zum Leben mit JavaScript. Lerne DOM-Manipulation, Event Handling und grundlegende Logik.

  • Grundlagen von JavaScript
  • Arbeiten mit dem DOM
  • Formularvalidierung und Event-Handling
